# Support — vianium-voip

Thanks for using **vianium-voip** — part of the [Vianium ecosystem](https://github.com/vianium).

This file lists the ways you can get help and the ways you can support the
project. Every Vianium repository carries an identical copy so the
process is consistent regardless of which library you arrived through.

---

## Getting help

| Need | Channel |
|---|---|
| Reproducible bug | [Open an issue](https://github.com/vianium/vianium-voip/issues/new/choose) with the **Bug report** template |
| Feature suggestion | [Open an issue](https://github.com/vianium/vianium-voip/issues/new/choose) with the **Feature request** template |
| Security disclosure | See [SECURITY.md](SECURITY.md) — do **not** open a public issue |
| Trademark / brand questions | See [TRADEMARKS.md](TRADEMARKS.md) where applicable |
| Commercial licensing (Tier 3 products only) | `hello@angelcareaga.com` |
| Other questions | [Open a discussion](https://github.com/vianium/vianium-voip/discussions) or email `hello@angelcareaga.com` |

Please search closed issues and discussions before opening a new one —
the answer to most "how do I…" questions is already documented in
[vianium-docs](https://github.com/vianium/vianium-docs).
